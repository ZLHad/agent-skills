# 输出文件规范

## 整理报告命名

```
整理报告-YYYYMMDD.md
整理报告-20250130.md
```

## 专题文档命名

```
学术写作错误手册.md（持续更新，保留历史版本在history/）
学术写作模式库.md（持续更新）
思维框架库.md（持续更新）
概念知识网络-[主题].md（按主题分文件）
```

## 索引文件命名

```
主题索引.md（每次全量重新生成）
标签索引.md（每次全量重新生成）
时间索引.md（每次全量重新生成）
文件清单-YYYYMMDD.md（带时间戳）
```

## 目录结构参考

```
[本地知识库路径]/
├── 原始记录/
│   ├── 学术写作/
│   ├── 学术探讨/
│   ├── 编程开发/
│   └── 其他/
│
├── 整理报告/
│   ├── 整理报告-20250130.md（本次生成）
│   └── 历史报告/
│
├── 专题整理/
│   ├── 学术写作错误手册.md
│   ├── 学术写作模式库.md
│   ├── 思维框架库.md
│   └── 概念知识网络.md
│
└── 索引/
    ├── 主题索引.md
    ├── 标签索引.md
    ├── 时间索引.md
    └── 文件清单.md
```
