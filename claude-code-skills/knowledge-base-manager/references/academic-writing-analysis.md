# 学术写作深度分析参考

本文档指导如何深度分析学术写作类知识库文件。

## 分析流程

### 1. 错误模式分析

#### 提取目标
从所有学术写作类文档中提取标记为 ⚠️ 的内容，这些是用户反复出现的错误。

#### 分类维度
按以下六个维度分类：
1. **语法错误**：主谓不一致、时态错误、单复数错误、冠词误用等
2. **词汇问题**：词汇选择不当、搭配错误、拼写错误等
3. **句式问题**：句子结构不清、修饰语位置不当、并列结构不对称等
4. **逻辑问题**：逻辑跳跃、论证不充分、因果关系不清等
5. **学术规范**：引用格式错误、术语使用不当、时态规范错误等
6. **风格问题**：过于口语化、冗余表达、表达不够学术等

#### 统计方法
对每个错误：
- **错误描述**：简洁描述错误类型
- **出现次数**：统计在多少篇文档中出现
- **首次出现**：最早的文档日期
- **最近出现**：最新的文档日期
- **改正状态**：
  - ✅ 已改正（近期文档中未再出现）
  - 🔄 有改善（出现频率降低）
  - ❌ 仍在出现（近期仍有）
- **典型案例**：错误示例 → 正确示例
- **记忆技巧**：帮助记住的方法
- **出现文档列表**：哪些文档中出现过

#### 排序规则
按出现频次从高到低排序，生成"高频错误Top10"或Top15。

#### 进步追踪
对比早期（前1/3文档）vs 近期（后1/3文档）：
- 计算错误率变化
- 识别已消失的错误（早期有，近期没有）
- 识别新出现的错误（早期没有，近期有）

---

### 2. 写作模式提取

#### 提取目标
从文档中提取标记为 💡 的可复用模式、成功的写作模板。

#### 分类维度
按论文章节分类：
1. **引言写作模式**
2. **文献综述模式**
3. **方法描述模式**
4. **结果呈现模式**
5. **讨论组织模式**
6. **结论撰写模式**
7. **摘要写作模式**

#### 每个模式记录
- **模式名称**：简洁命名
- **适用场景**：什么类型的论文/章节
- **结构描述**：清晰的结构框架
- **成功率**：如果多次使用，记录成功率
- **成功案例**：哪些文档中成功应用
- **注意事项**：使用时需要注意什么
- **示例模板**：具体的写作模板（如果有）

#### 合并相似模式
如果多个文档中提到相似的模式：
- 合并为一个模式
- 标注多次验证 ⭐⭐⭐
- 记录所有成功案例

---

### 3. 个人习惯档案

#### 句式偏好
- **高频句式**：统计经常使用的句式模板
- **有效句式**：标记为 ✅ 的句式
- **需避免句式**：标记为 ❌ 的句式

#### 连接词使用
- **高频连接词**：however, moreover, therefore等
- **使用恰当的场景**：记录正确使用
- **误用场景**：记录错误使用及改正

#### 时态把握
- **各章节时态偏好**：引言/方法/结果/讨论中习惯用什么时态
- **时态问题**：如果有时态混乱

#### 语态选择
- **主动/被动倾向**：更倾向使用哪种语态
- **需要调整的场景**：什么时候需要切换

#### 术语库
整理所有提到的专业术语：
- 中英文对照
- 使用场景
- 注意事项（如首字母大写、缩写规则）

---

### 4. 进步追踪

#### 对比维度
将所有文档按时间排序，分为：
- 早期（前1/3）
- 中期（中间1/3）
- 近期（后1/3）

#### 对比指标
1. **错误率变化**：
   - 早期平均每篇错误数
   - 近期平均每篇错误数
   - 变化百分比

2. **质量评分趋势**：
   - 如果文档中有质量评分，绘制趋势线

3. **技能掌握**：
   - 新掌握的写作技能（近期有，早期没有）
   - 巩固的技能（早期不熟练，近期熟练）

4. **问题消失**：
   - 哪些早期的问题现在已经不再出现

#### 输出格式
生成进步报告，包含：
- 整体进步概览
- 已改正的错误列表
- 新掌握的技能列表
- 仍需强化的方面
- 下阶段学习建议

## 处理原则

1. **数据驱动**：所有结论基于文档中的实际数据
2. **客观准确**：如实统计，不夸大也不遗漏
3. **可追溯**：每个结论都能追溯到具体文档
4. **面向改进**：不只指出问题，更要提供改进建议
5. **鼓励为主**：强调进步，同时指出需要改进的地方
