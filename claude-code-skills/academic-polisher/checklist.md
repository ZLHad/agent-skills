# 质量检查清单

在输出润色结果前，逐项自检：

## 语言检查
- [ ] 中文已全部转换为英文
- [ ] 时态使用正确
- [ ] 语态恰当（主动/被动）
- [ ] 用词学术化，避免口语
- [ ] 拼写和语法正确

## 逻辑检查
- [ ] 句子之间有过渡
- [ ] 因果关系清晰
- [ ] 论述完整（没有跳跃）
- [ ] 前后呼应

## 公式检查
- [ ] LaTeX 语法正确
- [ ] 符号定义清晰
- [ ] 上下标使用正确
- [ ] 算子格式规范（\min, \sum 等）
- [ ] 公式编号和引用正确

## 术语检查
- [ ] 专业术语规范
- [ ] 缩写首次出现给出全称
- [ ] 全文术语使用一致
- [ ] 变量命名一致

## 格式检查
- [ ] LaTeX 命令正确
- [ ] 标点符号规范
- [ ] 引用格式正确（\cite{}, \ref{}）
- [ ] 符合期刊模板要求
