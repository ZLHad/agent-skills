# Interactive Q&A Templates 交互问答模板

## 快速启动问答

### 简洁模式（推荐）

```
我来帮您生成绘图Prompt。请先告诉我：

1. **图片类型**：
   □ 框架流程图  □ 系统场景图  □ 算法架构图
   □ 时序阶段图  □ 2.5D场景图  □ 环绕布局图
   □ 其他（请描述）

2. **核心内容**：请简述或上传文档/参考图

3. **风格偏好**（可选，有默认值）：
   - 比例：16:9 / 4:3 / 1:1
   - 配色：学术蓝灰 / 多彩清新 / 自定义
   - 语言：中文 / 英文
```

### 极简模式

```
请描述您需要的图片内容，我会根据内容自动选择合适的布局和风格。

如有参考图片请一并上传。
```

## 按需追问清单

根据用户初始输入的完整程度，选择性追问：

| 信息缺失 | 追问方式 |
|----------|----------|
| 图片类型不明 | "这张图主要展示：系统部署场景？算法流程？数据处理？" |
| 布局不明确 | "您倾向于：左右流水线？上下分层？中心辐射？" |
| 元素关系不清 | "这些模块之间是：顺序处理？并行处理？层级关系？" |
| 风格无偏好 | 使用默认值，不追问 |
| 有参考图 | "我注意到参考图的风格是[X]，是否采用类似风格？" |

---

## 默认值设定

| 参数 | 默认值 | 触发条件 |
|------|--------|----------|
| 比例 | 16:9 | 未指定 |
| 配色 | 学术蓝灰 | 未指定 |
| 语言 | 根据对话语言 | 未指定 |
| 风格 | 矢量扁平 | 始终 |
| 图注 | 生成 | 未明确拒绝 |
| 图例 | 根据复杂度自动判断 | 未指定 |

## 类型识别关键词

| 用户说法 | 推断类型 |
|----------|----------|
| "网络拓扑"、"部署"、"场景"、"卫星/无人机/基站" | 系统场景图 |
| "算法"、"网络结构"、"Actor-Critic"、"训练" | 算法架构图 |
| "流程"、"步骤"、"模块"、"处理" | 框架流程图 |
| "时间轴"、"阶段"、"周期"、"时隙" | 时序阶段图 |
| "城市"、"立体"、"俯视"、"3D" | 2.5D场景图 |
| "功能模块"、"应用"、"环绕"、"中心" | 环绕布局图 |

## 参考图分析要点

当用户上传参考图时，快速识别：

1. **布局结构**：分层/流水线/环绕/混合
2. **视觉风格**：扁平/立体/手绘/写实
3. **配色方案**：单色系/多彩/渐变
4. **元素类型**：图标/框图/场景/图表
5. **标注密度**：密集/适中/简洁
