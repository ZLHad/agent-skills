## 四、布局间距

### 比例参考
| 比例 | 适用 |
|------|------|
| 16:9 | PPT、宽屏展示 |
| 4:3 | 传统、方正布局 |
| 2:1 | 横向流程图 |
| 1:1 | 对称结构 |

### 元素间距原则
- 同组元素：紧凑
- 不同组：明显间隔
- 主次分明：主要元素大且居中

## 五、视觉风格关键词

生成Prompt时可使用的风格描述词：

**整体风格**
- vector style（矢量风格）
- flat design（扁平设计）
- minimal（简约）
- clean lines（线条清晰）
- professional（专业）
- academic illustration（学术插图）

**立体感**
- 2D flat（纯平面）
- 2.5D isometric（等轴测）
- subtle shadows（轻微阴影）
- layered（分层感）

**配色描述**
- blue-gray palette（蓝灰配色）
- vibrant colors（鲜明色彩）
- soft gradients（柔和渐变）
- high contrast（高对比）

**避免使用**
- realistic（写实）
- photographic（照片级）
- 3D render（3D渲染）
- glossy（光泽感）
- complex textures（复杂纹理）
