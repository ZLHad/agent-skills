# Academic Figure Prompt — 学术插图 Prompt 生成

为科研论文和学术汇报生成高质量、完整自包含的插图绘制 Prompt，可直接喂给生图 AI 使用。

## 核心特性

- **完整自包含**：生成的 Prompt 让生图 AI 无需额外信息即可正确生成
- **技术准确**：公式、参数、文字正确表达学术/技术含义
- **矢量图风格**：扁平化、简洁、专业，避免写实/炫酷风格
- **无元信息污染**：坐标、区域编号、尺寸标注等不出现在最终图像中

## 支持图片类型

| 类型 | 说明 |
|------|------|
| 框架流程图 | 模块化、左右/上下流水线布局 |
| 系统场景图 | 网络拓扑、设备部署、空间关系 |
| 算法架构图 | 神经网络、强化学习、数据流 |
| 时序/阶段图 | 时间轴、多阶段流程 |
| 2.5D/等轴测图 | 立体城市、网络场景 |
| 环绕/辐射图 | 中心场景 + 周围模块 |
| 对比/并列图 | 多方案对比 |

## 工作流程

```
Phase 1: 交互式信息收集（图片类型、核心内容、配色、比例等）
   ↓
Phase 2: 生成完整 Prompt（整体设计 + 详细描述 + 连接与数据流 + 文字清单 + 配色方案 + 约束）
   ↓
Phase 3: 迭代优化（根据反馈调整）
```
