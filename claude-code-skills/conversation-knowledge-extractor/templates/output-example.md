# 知识库文档输出示例

## 文件命名规范

```
knowledge-YYYYMMDD-{主题简称}-{序号}.md
```

**示例**：
- `knowledge-20250129-academic-writing-01.md`
- `knowledge-20250129-ml-discussion-01.md`
- `knowledge-20250129-python-debug-01.md`

---

## 分级展示说明

### 第一层：执行摘要
- 快速浏览版本
- 200-400字
- 让用户1分钟内了解对话核心价值

### 第二层：详细内容
- 完整的知识提取
- 3000-5000字（学术写作）
- 2000-3000字（学术讨论）
- 1500-2500字（技术任务）
- 按模板结构化组织

### 第三层：行动指南
- 可执行的建议
- 明确的下一步
- 关联的知识点

## 视觉标记系统

### 重要性标记
- ⭐⭐⭐ 最高优先级/核心知识点
- 💡 重要洞察/突破性认知
- ⚠️ 需要强化记忆/反复出现的错误
- 🔄 认知演进点
- 📌 个人偏好/习惯

### 状态标记
- ✅ 已解决/已掌握
- ⏸️ 部分解决/进行中
- ❓ 待解决/需进一步探索
- 🐛 Bug/问题
- 🔍 需要深入研究

### 内容类型标记
- 📋 总结/清单
- 📚 知识/理论
- 💻 代码/技术
- 🎯 行动/建议
- 🔗 关联/链接
